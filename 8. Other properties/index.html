Другие свойства

  • margin, padding

    Добавление отступов и полей
    CSS: 
                    
.text{
/*Отступ*/
padding: 10px; /*отступ со всех сторон*/
padding: 0 10px; /*верх-низ, право-лево*/
padding: 0 10px 20px;/*верхнее, лево-право, нижнее*/
padding: 0 10% 20% 5%;/*сверху, справа, снизу, слева*/
/*Поле*/
margin: 10px;/* со всех сторон*/
margin: 0 10px;/*верх-низ, право-лево*/
margin: 0 10px 20px;/*верхнее, лево-право, нижнее*/
margin: 0 10px 20px 5px;/*сверху, справа, снизу, слева*/
}
    Нюансы
    • Если сталкиваются верхнее и нижнее поле разных элементов, то они не суммируются. ВЫбирается размер найбольшего поля, а меньшее удаляется. Хочешь сложения- используй отступ либо границу.
    • Добавление к строчному элементу (кроме <img>) верхнего, нижнего поля или отступа не изменит высоту элемента. Для элемента нужно применить: display: inline-block;

  • display

    Строчные, блочные, блочные в ряд
    CSS: 
                    
display: block;/*делает элемент блочным*/
display: inline;/*делает элемент строчным*/
display: inline-block;/* убирает разрывы до и после, добавляет поля ввреху и снизу*/
display: none;/*скрывает элементы*/

  • border

    Добавление границы к элементу
    HTML: 
                    
<div class="b-solid">solid</iv>
<div class="b-dashed">dashed</ltiv>
<div class="b-dotted">dotted</div>
<div class="b-double">double</div>
<div class="b-groove">groove</div>
<div class="b-inset">inset</div>
<div class="b-outset">outset</div>
<div class="b-ridge">ridge</div>
<div class="b-none">none</div>
<div class="b-hidden">hidden</div>
    CSS: 
                    
.b-solid {
	border: 6px solid rgb(255, 0, 0);
}

.b-dashed {
	border: 6px dashed rgb(255, 0, 0);
}

.b-dotted {
	border: 6px dotted rgb(255, 0, 0);
}

.b-double {
	border: 6px double rgb(255, 0, 0);
}

.b-groove {
	border: 6px groove rgb(255, 0, 0);
}

.b-inset {
	border: 6px inset rgb(255, 0, 0);
}

.b-outset {
	border: 6px outset rgb(255, 0, 0);
}

.b-ridge {
	border: 6px ridge rgb(255, 0, 0);
}

.b-none {
	border: 6px none rgb(255, 0, 0);
}

.b-hidden {
	border: 6px hidden rgb(255, 0, 0);
}
    Результат:
    solid
    dashed
    dotted
    double
    groove
    inset
    outset
    ridge
    none
    hidden
    Нюансы
    • Последовательность указания свойств неважна
    • Можно использовать свойства отдельные для сторон: border-top, border-bottom, border-left, border-right
    • Можно указывать отдельные свойства: border-color, border-width, border-style
    • Отдельные стороны и отдельные свойства: border--bottom-color
    • Можно сокращенно задавать значения для отдельных свойств: border-color: red, blue, gold, grean; border-width: 10px, 5px, 12px, 20px:

  • background-color

    Фоноввый цвет
    HTML: 
                    
<div class="bg-c1">background-color: rgb(247,226,155);</div>
<div class="bg-c2">background-color: rgba(9, 218, 19, 0.1);</div>
<div class="bg-c3">border: 6px dotted rgb(255, 0, 0); background-color: rgba(31, 8,	230, 0.1);;</div>
<div class="bg-c3">border: 6px dotted rgb(255, 0, 0); background-color: rgba(31, 8, 230, 0.1); background-clip: padding-box;;</div>
    CSS: 
                    
.bg-c{
	background-color: rgb(247,226,155);/*заливает фон цветом*/;
} 
.bg-c2 {
	background-color: rgba(9, 218, 19, 0.1);
}

.bg-c3 {
	border: 6px dotted rgb(255, 0, 0);
	background-color: rgba(31, 8, 230, 0.1);
}
.bg-c4 {
	border: 6px dotted rgb(255, 0, 0);
	background-color: rgba(31, 8, 230, 0.1); 
	background-clip: padding-box;
}
    Результат:
    background-color: rgb(247,226,155);
    background-color: rgba(9, 218, 19, 0.1)
    border: 6px dotted rgb(255, 0, 0); background-color: rgba(31, 8, 230, 0.1);
    border: 6px dotted rgb(255, 0, 0); background-color: rgba(31, 8, 230, 0.1); background-clip: padding-box;
    Нюансы
    • Точечная и пунктирная граница сливается с фоновым цветом, это можно обойти используя свойство background-clip: padding-box;
