Каскадность

Каскадность - это ряд правил определяющих, какие именно свойства стилей применяются к элементам веб-страницы.
    Правила
  1. Превосходство ближайших предков. Стиль ближайшего предка приоритетней дальнего.
    Пример кода 
    						
<div class="block1"> 
	<p class="p1"> 
		<strong> 
			Цвет
		</strong> 
	</p> 
</div>
    CSS-код 
    							
.block1 {
	color: red;
}
.p1 {
	color: blue;
}
    Результат

    Цвет

  2. Преимущество непосредственно примененного стиля
    Пример кода 
    						
<div class="block2">
	<p class="p2">Цвет</p>
</div>
    CSS-код 
    							
.block2 {
	color: blue;
}
.p2 {
	color: red;
}
    Результат

    Цвет

  3. Имеют преимущество стили самого близкого по отношению к форматированному элементу, самого специфичного стиля.
      Определение приоритетов (чем больше, тем выше приоритет)
    • Селектор тегов имеет 1 УЕ
    • Класс имеет 10 УЕ, а ID - 100 УЕ
    • Строчный стиль написания стилей (в коде HTML) имеет 1000 УЕ
    • Следует помнить, что данные значения могут складываться: div p {} равен 2 УЕ
  4. Если приоритет одинаковый то результатом конфликта выберется последний(нижний)
  5. Если внутренние стили HTML страницы (тег style) идут после подключения внешних стилей, то они будут приоритетней.
  6. При наличии свойства !important касдадность игнорируется
    Пример кода 
    						
<div class="block3">
	<p class="p3">Цвет</p>
</div>
    CSS-код 
    							
.block3 .p3 { 20УЕ
	color: red;
}

.p3 { 10УЕ
	color: blue !important;
}
    Результат

    Цвет
